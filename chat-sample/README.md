# chat-sample

## about directory structure   
```
- src
    - assets
    - components  <- routerファイルに書かれているmain vue
    - environment
    - repositories
    - models
    - router
    - store
    - views <- routerに書かれておらず、vueファイル内で呼ばれるvueファイル
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
